# Portfólio — Kaenny Ribeiro Granja

Versão React migrada da versão HTML/CSS/JS original (branch `main`).

## Stack
- React 18
- Vite 5

## Estrutura
```
src/
├── components/  # Header, Hero, Sobre, Jornada, Habilidades, Projetos, Contato, Footer
├── data/        # projetos.js (arrays renderizados dinamicamente)
├── styles/      # index.css
├── App.jsx
└── main.jsx
```

## Scripts
```bash
npm install
npm run dev       # http://localhost:5173
npm run build
```
